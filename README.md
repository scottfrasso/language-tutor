# Polish Language Learning Scenarios

Realistic A1/A2 Polish conversation practice scenarios for learners living in Poland.

## What This Is

A collection of 26+ realistic Polish conversation scenarios designed to help you practice everyday situations you'll encounter living in Poland. Unlike textbook dialogues, these scenarios include multiple possible responses and variations - just like real conversations.

## Features

- **A1 & A2 Levels**: Progressive difficulty from beginner to elementary
- **Realistic Dialogues**: Multiple shopkeeper responses, upselling attempts, mishearings
- **Pronunciation Guides**: Phonetic pronunciation for every Polish phrase
- **Cultural Notes**: Polish customs, formality levels, shopping norms
- **Polish + English**: Full translations with English meanings
- **Practical Locations**: Auchan, Kaufland, piekarnia, apteka, farmers markets, neighbors

## Scenario Categories

### A1 (Beginner)
- Grocery Basics (Auchan, Kaufland)
- Specialty Shops (Piekarnia, Cukiernia, Apteka, Farmers Market)
- Social Situations (Elevator, Hallway, Meeting Neighbors)

### A2 (Elementary)
- Grocery Intermediate (Recommendations, Dietary Needs, Negotiating)
- Specialty Shops Advanced (Special Orders, Describing Symptoms)
- Social Intermediate (Small Talk, Problem Solving)

## How to Use

1. **Choose Your Level**: Start with A1 if you're a beginner
2. **Pick a Scenario**: Choose a situation you'll encounter soon
3. **Study the Vocabulary**: Review the key words table
4. **Practice the Dialogue**: Read through multiple variations
5. **Focus on Pronunciation**: Use the phonetic guides (CAPS = stress)
6. **Try it in Real Life**: Use what you've learned!

## Pronunciation Guide

- **CAPS**: Stressed syllable (e.g., po-PRO-sheh)
- **Hyphens**: Syllable breaks
- **Phonetics**: English approximations in parentheses

Example: "Dziękuję" = (jen-KOO-yeh)

## Adding More Scenarios

Use the TEMPLATE files in each category to create new scenarios:
- `scenarios/A1/specialty-shops/TEMPLATE-A1.md`

Just copy, fill in the dialogue, and commit!
