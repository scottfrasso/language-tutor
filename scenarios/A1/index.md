# A1 Level - Beginner Polish Scenarios

Welcome to A1 level! These scenarios use basic vocabulary (8-12 words) and simple sentence structures. Perfect for beginners who are just starting to speak Polish.

## What is A1?

- **Vocabulary**: 8-12 words per scenario
- **Grammar**: Simple present tense, basic requests
- **Interactions**: 2-4 short exchanges
- **Goals**: Single, clear objective (buy bread, greet neighbor)

## Tips for A1 Learners

1. **Start simple**: Use the "Easiest Version" provided in each scenario
2. **Memorize key phrases**: "Poproszę" (I'd like), "Ile kosztuje?" (How much?), "Dziękuję" (Thank you)
3. **Pronunciation matters**: Focus on the phonetic guides (CAPS = stress)
4. **Don't worry about mistakes**: Poles appreciate any attempt to speak Polish!
5. **Repeat, repeat, repeat**: Practice the same scenario multiple times
